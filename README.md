# Dingle-goodies

딩글(Dingle) 앱의 **문의 탭 > 딩글 소식**이 읽어가는 공식 콘텐츠 저장소입니다.
여기에 push하는 것만으로 전 유저에게 배포됩니다 (앱 업데이트 불필요).

| 파일 | 내용 |
|---|---|
| `notices.json` | 수시 공지사항 (제목/본문/이미지, `pinned`로 상단 고정) |
| `update-notes.json` | 버전별 업데이트 공지 |
| `manifest.json` | 공식 자료실 목록 (스티커팩/테마) |
| `notices/`, `thumbs/`, `themes/`, `packs/` | 이미지·자산 파일 |

작성 규칙과 항목 형식은 메인 레포의
[docs/GOODIES_REPO_GUIDE.md](https://github.com/binglehaepi/Dingle/blob/main/docs/GOODIES_REPO_GUIDE.md) 참고.

⚠ **보호(작가) 에셋의 원본 파일은 절대 이 공개 레포에 올리지 마세요** — 패킹 CLI로 암호화한 .dga만 업로드합니다.
